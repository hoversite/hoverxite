# Contributing to Hoverxite

Thanks for helping build the open-source site agent.

## Before you start

Read [VISION.md](./VISION.md) and [OPEN-CORE.md](./OPEN-CORE.md). Contributions
that pull in private cloud-only features, secrets, or competitor research will
be rejected.

## Development

```bash
git clone https://github.com/hoversite/hoverxite.git
cd hoverxite
npm install
npm run dev
```

## Pull requests

1. Branch from `main`.
2. Keep changes focused.
3. Add a changelog bullet for user-visible behavior.
4. Describe *what* and *why* in the PR body.
